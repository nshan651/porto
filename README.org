My Mimimum Viable (Development) Container. These are all the tools I need to be productive, all within a portable Apline Linux image.
[[./green.png]]
This is a "literate config," which means everything here in =README.org= is written to a Dockerfile using [[https://orgmode.org/manual/Extracting-Source-Code.html#Functions-1][org-babel-tangle]].

* Basic Dependencies

#+begin_src dockerfile :tangle "Dockerfile"
# Use a minimal Alpine Linux image as the base
FROM alpine:latest

# Set the working directory
WORKDIR /home/nick

# Install base dependencies
RUN apk --update --no-cache add \
    build-base \
    curl \
    gcc \
    git \
    musl-dev
#+end_src

* Setup User and $HOME

#+begin_src dockerfile :tangle "Dockerfile"
# Create a new user
RUN locale-gen en_US.UTF-8 && \
    adduser --quiet --disabled-password --shell /bin/zsh --home /home/nick --gecos "User" nick && \
    echo "nick:p@ssword1" | chpasswd &&  usermod -aG sudo nick
#+end_src

* Development Tools
Now we're going to want a shell, a text editor, and maybe some other useful CLI programs. Pick your favorites.
#+begin_src dockerfile :tangle "Dockerfile"
# Some basic development tools
RUN apk --update --no-cache add \
    zsh \
    bash \
    neovim \
    emacs \
    tmux \
    lf \
    ripgrep
#+end_src
